# Mega Man 3 Poptracker Pack

Very early work in progress, just getting my feet wet with Poptracker. Feel free to use, abuse, etc. You can install it by adding the ZIP file in the release to your installed Poptracker.

## Known Issues

Manual click tracking isn't working for Break Man and Wily Stages yet.
